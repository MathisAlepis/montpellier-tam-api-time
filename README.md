# montpellier-tam-api

## Example

Querying https://montpellier-tam-api.vercel.app/api/query?stop_name=ANTIGONE&trip_headsign=MOSSON will retrieve all courses from ANTIGONE to MOSSON.

## Available fields

- course
- stop_code
- stop_id
- stop_name
- route_short_name
- trip_headsign
- direction_id
- departure_time
- is_theorical
- delay_sec
- dest_ar_code
